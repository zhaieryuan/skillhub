# 提升到全局

优秀的团队技能可以申请提升到全局空间，供全企业使用。

## 提升前提

- 技能在团队空间已发布
- 申请人是技能 owner 或命名空间 ADMIN
- 技能没有待审核的提升申请

## 申请提升

1. 进入团队技能详情页
2. 点击"提升到全局"
3. 填写申请说明
4. 提交申请

## 审核流程

1. 平台管理员收到提升申请
2. 审核技能质量和适用性
3. 审核通过后：
   - 在全局空间创建新技能
   - 保留原团队技能
   - 记录来源追溯关系

## 提升后

- 全局空间的新技能独立管理
- 原团队技能继续存在
- 两者版本不自动同步
- 如需同步需手动操作
