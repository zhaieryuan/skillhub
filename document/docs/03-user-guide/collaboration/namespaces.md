# 团队命名空间

## 加入命名空间

需要命名空间管理员邀请才能加入团队命名空间。

## 命名空间角色

### MEMBER
- 可查看命名空间内所有技能
- 可发布技能（需审核）
- 可收藏和评分

### ADMIN
- 所有 MEMBER 权限
- 可审核技能发布
- 可管理成员
- 可编辑命名空间信息

### OWNER
- 所有 ADMIN 权限
- 可转让所有权
- 可归档命名空间

## 技能可见性

| 可见性 | 说明 |
|--------|------|
| `PUBLIC` | 所有人可见，匿名可下载 |
| `NAMESPACE_ONLY` | 仅命名空间成员可见 |
| `PRIVATE` | 仅 owner 和命名空间 ADMIN 可见 |
