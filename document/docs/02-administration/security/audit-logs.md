# 审计日志

SkillHub 记录所有关键操作的审计日志，满足企业合规要求。

## 审计范围

记录的操作包括：
- 技能发布、下载、删除
- 审核通过、拒绝
- 用户登录、登出
- 权限变更
- 命名空间管理
- 配置变更

## 审计日志查询

通过管理后台审计日志页面或 Admin API 查询。

## 日志字段

- 操作时间
- 操作用户
- 操作类型
- 目标资源
- 客户端 IP
- User-Agent
- 请求 ID
- 详细信息
